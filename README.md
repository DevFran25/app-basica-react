# VotApp Frontend

Aplicación frontend de VotApp, construida con **React**, **Vite** y **TailwindCSS**.

## Instalación

```bash
Clonar el repositorio:
git clone <URL_DEL_REPOSITORIO>
cd votapp-frontend

2.Instalar dependencias:
   npm install

3.Ejecutar la app en modo desarrollo:
npm run dev

src/
  ├─ components/        # Componentes reutilizables (UI, Cards, Inputs, Confirmacion)
  ├─ pages/             # Páginas principales (Home, Login, Dashboard)
  ├─ services/          # Funciones para consumir APIs (api.js)
  ├─ App.jsx
  ├─ main.jsx
  └─ assets/            # Imágenes, logos y recursos estáticos
